Cosa sono le API?

API = Application Programming Interface
Un “contratto” tra due software per scambiarsi dati.

Analogia: il cameriere (= API) porta il vostro ordine alla cucina (= server) e vi riporta il piatto (= risposta). Non entrate mai in cucina.
